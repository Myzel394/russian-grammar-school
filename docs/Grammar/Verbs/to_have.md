# To have

To say "I have [...]" in Russian, you literally say: "At me is [...]"

* У **меня** есть [...] (I have ...)
* У **тебя** есть ... (You have ...)
* У **него** есть [...] (He has ...)
* У **нее** есть [...] (She has ...)
* У **нас** есть [...] (We have ...)
* У **вас** есть [...] (You (plural / formal) have ...)
* У **них** есть [...] (They have ...)
