# Nominative Case

* Base form of a noun
* Change the ending only when forming the plural

## Plural

| Masculine                 | Feminine                                              | Neuter   |
|---------------------------|-------------------------------------------------------|----------|
| (ends in consonant) -> -ы |                                                       |          |
| -й -> -и                  | -я -> -и                                              | -о -> -а |
| -ь -> -и                  | -ь -> -и                                              | -е -> -я |
|                           | (Г, К, Х, Ж, Ч, Ш, Щ) + -а -> -и (otherwise) -а -> -ы |          |
