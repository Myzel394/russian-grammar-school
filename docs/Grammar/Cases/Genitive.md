# Genitive

## When to use the genitive case

1. When you're talking about **amounts**
    * One cat, two cats, five cats
    * Один кот, два кота, пять котов
2. For **negations**
    * There is no cat
    * Нет кота
3. When you're talking about **possession**
    * This is my cat
    * Это мой кот
    * My friend's cat
    * моего друга Кот 

## Nouns

### Singular

| Masculine                     | Feminine                          | Neutral                 |
|-------------------------------|-----------------------------------|-------------------------|
| (ends in consonant) -> add -а | (Г, К, Х, Ж, Ч, Ш, Щ) + -а -> -и (otherwise) -а -> -ы |                         | 
| -й -> -я                      | -ь -> -и                          | о -> а                  |
| -ь -> -я                      | -я -> -и                          | е -> я                  |
|                               |                   | мя -> мени                       |

| Masculine                  | Feminine                                              | Neuter   |
|----------------------------|-------------------------------------------------------|----------|

| -ь -> -ей                  | -я -> -ь                                              | -е -> -ей |

## Pronouns

I have = У меня есть
You have = У тебя есть
He has = У него есть
She has = У неё есть
We have = У нас есть
You have (plural) = У вас есть
They have = У них есть

## Examples

I don't have a cat

У меня нет кота

(У меня, нет, кот)
